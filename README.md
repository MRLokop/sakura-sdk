# Sakura SDK

## Install
```bash
npm i sakura-sdk --save
```

## Example
```javascript
const { Sakura } = require("sakura-sdk");

const bot = new Sakura({
    key: "YOUR_BOT_KEY"
});
```
And it prefectly work with TypeScript too
```typescript
import { Sakura } from "sakura-sdk";

...
```
